Pasos para ejecutar la prueba E2E realizada con Selenium:

1. Tener instalado python en el equipo que ejecutara la prueba, puede obtener mas informacion de como instalarlo
aqui: https://www.python.org/downloads/

2. En caso de no tener pip instalado en el equipo realizarlo mediante el comando: python get-pip.py

3. Instalar pytest usando el siguiente comando: pip install pytest

4. Instalar selenium usando el siguiente comando: pip install selenium

5. Instalar webdriver-manager usando el siguiente comando: pip install webdriver-manager

6. Instalar pytest-html usando el siguiente comando: pip install pytest-html

7. Instalar requests usando el siguiente comando: pip install requests

8. Ubicar el directorio de trabajo: testAutomation/automation_e2e

9. Ejecutar la prueba e2e con el siguiente comando: pytest test_purchase_flow.py
